# 泉州考察日程和主题

泉州=刺桐城

考察时间：2024.3.28— 2024.4.1

考察线索：泉州-佛国之城，何以多“异端”

考察费用：19800+2500=22300

总问题：

- 为何在极尽繁华的同时成为历史上的一个异数？
- 人、商业与文化的逻辑如何交汇，共生为古代中国的一个另类样本。
- 一同解析中国古代另类商业文明的典范城市，探寻世界潮流大势的风云变幻逻辑所在。

    - 开元寺

    - 市舶司遗址
